Welcome to Tech Cache's documentation!
======================================


Detta program är till att man ska enklare ha koll på alla komponenter som
man har i sin workshop vid t.ex 3D printings projekt/ robotik eller liknande.

Så i detta program kan man spara och ha koll på hur mycket man har av varje
ha beskrivningar/manualer, länkar till inköp sidor osv.

Funkionalitet
-------------

* produkternas info i en tabell
* redigera/updatera info
* sortering via attributer
* sök funktionen
* exportera/importera info 
* alerts för low stock "dags att köpa in"
